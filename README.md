# Memo Flashcards

Stack: Nuxt 3 + TypeScript + Tailwind + Pinia + Capacitor + SQLite

Installation

```
npm install
```

Dev

```
npm run dev
```

Build

```
npm run build
```

Tests

Unit

```
npm run test:unit
```

E2E

```
npm run test:e2e
```

Capacitor commands

```
npx cap add ios
npx cap add android
npx cap sync
npx cap open ios
npx cap open android
```
